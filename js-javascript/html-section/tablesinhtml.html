Tables in HTML

Tables are used to display data in a structured format.

They consist of rows and columns, where each cell can contain text, images, or other HTML elements.

In this example, we have a simple table with three columns: Name, Age, and City.

Each row represents a different person with their respective details.

Tables can be styled using CSS to enhance their appearance.

Table Example

This example shows how to create a basic table in HTML.

Tables can be styled using CSS to enhance their appearance.

Use the <table>, <thead>, <tbody>, and <tr> elements to structure your table.

You can also use <th> for header cells and <td> for data cells.

Make sure to include appropriate styling to improve readability.

For example, you can add borders, padding, and background colors to the table cells.

Tables are commonly used in web applications to display data in a tabular format.

They can also be used for layout purposes, although this is less common in modern web design.

Remember to keep your tables accessible by using proper markup and semantic elements.

Consider using ARIA roles and attributes to enhance accessibility for users with disabilities.

Overall, tables are a powerful tool for organizing and presenting data on the web.

Use them wisely to create clear and informative layouts.
