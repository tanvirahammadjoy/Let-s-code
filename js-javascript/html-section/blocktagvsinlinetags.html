Block vs Inline Elements

In HTML, elements can be categorized as block-level or inline elements.

Block-Level Elements

Block-level elements take up the full width available and start on a new line.

Examples include:

  • <div>
  • <h1> to <h6>
  • <p>
  • <form>
  • <table>
  • <section>
  • <article>

Inline Elements

Inline elements only take up as much width as necessary and do not start on a new line.

Examples include:

  • <span>
  • <a>
  • <img>
  • <strong>
  • <em>
  • <small>
  • <cite>
  • <q>
  • <abbr>
  • <code>
  • <kbd>
  • <samp>
  • <var>
  • <time>
  • <mark>
  • <sub>
